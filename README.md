# TypeCmd
## 简介
* **TypeCmd**是一个模拟命令行，使用Rust写成
## 特性
* **丰富的输出**: 所有输出使用彩色形式
* **强大的历史支持**: 提供类似bash的历史记录
* **变量支持**: 提供类似var somevar value形式的变量赋值
## 安装
* **预构建版本**: 前往Github Release下载即可
* **手动编译**:
  1. 克隆源码
  2. 运行make命令
  3. 以超级用户权限运行make install命令，它将把typecmd安装到/usr/local/bin
## 运行
* **添加好PATH**后，输入`typecmd`即可运行
